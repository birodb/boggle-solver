## boggle-solver
A _functional_ [xquery 3.1](https://www.w3.org/XML/Group/qtspecs/specifications/xquery-31/html/xquery-31-diff.html) implementation to solve the [boggle game](https://en.wikipedia.org/wiki/Boggle).
